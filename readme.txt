==================================================
THE HARDEST GAME IN THE WORLD (DOCTOR WHO): README
==================================================

- You are the doctor (blue block). 
- You must AT ALL COSTS, avoid the evil grey daleks and make it back to your TARDIS to save River Song in an alternate universe!
- Use the arrow keys to move the doctor.
- Hit select at any point to restart the game.

- ENJOY! DO NOT GET EXTERMINATED!
